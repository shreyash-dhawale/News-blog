Daily News and Blogs API Documentation

Introduction

Welcome to the Daily News and Blogs API documentation. This API provides access to news articles and blog posts. Users can view, create, update, and delete blogs using the provided endpoints.

Authentication

This API uses HTTP Basic Authentication for user authentication. To access the API endpoints, you need to provide your username and password in the following format:
https://username:password@api_url/endpoint

For Testing Purpose use:

UserName = “admin”

Password = “password”

API Endpoints

1. Get Blogs List

  • Method: GET
  • Endpoint: /api/blogs/

Example using curl

curl -X GET https://admin:password@dailynewsblogs.pythonanywhere.com/api/blogs/

Description: This endpoint allows you to retrieve a list of existing blogs.

2. Create a Blog

  • Method: POST
  • Endpoint: /api/blogs/create/
  • Parameters:

title (string): The title of the blog.

body (string): The content of the blog.

Example using curl

curl -X POST https://admin:password@dailynewsblogs.pythonanywhere.com/api/blogs/create/ -d "title=Api Testing" -d "body=Api Endpoint Testing"

Description: This endpoint allows you to create a new blog with the specified title and content.

3. Update a Blog

  • Method: PUT
  • Endpoint: /api/blogs/update/<blog_id>/
  • Parameters:

title (string): The updated title of the blog.

body (string): The updated content of the blog.

Example using curl

curl -X PUT https://admin:password@dailynewsblogs.pythonanywhere.com/api/blogs/update/8/ -d "title=Api Testing" -d "body=Checking Update Endpoint"

Description: This endpoint allows you to update an existing blog by specifying the blog's ID and providing the updated title and content.

4. Delete a Blog

  • Method: DELETE
  • Endpoint: /api/blogs/delete/<blog_id>/

Example using curl

curl -X DELETE https://admin:password@dailynewsblogs.pythonanywhere.com/api/blogs/delete/8/

Description: This endpoint allows you to delete an existing blog by specifying the blog's ID.

Response

The API returns JSON responses with appropriate status codes to indicate the success or failure of each request.

Conclusion

This API provides the functionality to interact with blogs, allowing you to retrieve, create, update, and delete blog posts. Use the provided endpoints as documented above to perform these actions.

If you encounter any issues or have questions, please contact our support team for assistance.

Thank you for using the Daily News and Blogs API!
